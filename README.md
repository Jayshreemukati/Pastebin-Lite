# Pastebin-Lite

A lightweight Pastebin-like web application built using Node.js, Express, and MySQL.

## Features
- Create text pastes
- Generate shareable links
- View pastes via public URL
- Track number of views
- Persistent storage using MySQL

## Tech Stack
- Node.js
- Express.js
- MySQL
- HTML + JavaScript

## Run Locally

```bash
npm install
npm start
